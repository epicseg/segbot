General Commands [36]

Command Description
.help Sends the link to this website
.say Makes the bot say the text of your choice.
.avatar (user) Sends the mentioned users avatar.
.whois (user) Displays discord information of the mentioned user
.mute (user) Mutes the mentioned user.
.unmute (user) Unmutes user.
.ban (user) Bans mentioned user.
.unban user#0000 Unbans the user. Make sure to put the user and tag.
.nuke Deletes all messages in the channel.
.serverinfo Displays info about a server.
.bitcoin Shows the current price of bitcoin.
.slowmode (seconds) Sets a custom slowmode delay for the channel.
